このセットはpak128japanの産業を拡張するための工場を作るものです。

1900年ごろから重工業で遊べるようにしています。

自動車の製造や家電の製造などで工業国日本を再現できればいいなと思っています。
(まだちょっと乱暴な部分があるので、産業の追加は上級者向きです。)
